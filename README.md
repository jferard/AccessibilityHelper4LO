## Configuration
Needs `pytest` and `flake8`.

```
$ python3 -m pip install -r requirements.txt
$ sudo touch /var/log/AccessbilityHelper4LO.log
$ sudo chmod 666 /var/log/AccessbilityHelper4LO.log
 ```

## Installation
```
$ python3 py4lo_extension_builder.py install
```
